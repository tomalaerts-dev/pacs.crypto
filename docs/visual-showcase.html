02 · Lifecycle map

One end-to-end lifecycle, at a glance

The same path the video walks through: bank instruction enters via Travel Rule, the sending VASP quotes and broadcasts, the chain settles, the receiving VASP confirms, and reporting and remediation hang off the end.

  1. Bank Originating bank / treasury submits travel-rule + instruction implemented
  2. SVASP Sending VASP /travel-rule, /instruction implemented
  3. Travel Rule Travel Rule record pacs.008-aligned identity + remittance implemented
  4. Quote Instruction quote POST /instruction/quote implemented
  5. Instruction Instruction submit linked travel_rule_record_id implemented
  6. Chain Chain / finality PENDING → BROADCAST → CONFIRMING → FINAL implemented
  7. RVASP Receiving VASP Travel Rule callback / ack implemented
  8. Status Status / reporting / evidence camt.054 / camt.052 / camt.053 / camt.025-like implemented
  9. Return Return case pacs.004 vocabulary preserved implemented
  10. Reverse Reversal path pacs.007-aligned implemented
  11. Cancel Pre-broadcast cancel camt.056 analogue only partial
  12. E&I Investigation family camt.026 / camt.027 / camt.087 — TBD draft / not locked

04 · ISO 20022 alignment

Where the ISO 20022 vocabulary lands

Industry vocabulary is preserved where it helps reviewers recognise intent quickly, and deliberately narrowed where the public-chain context makes the namesake misleading.

pacs.004

Return vocabulary

Post-settlement return cases keep pacs.004 wording. It is recognised industry terminology and reads the way reviewers already expect.

implemented
pacs.007

Reversal / remediation path

/return-cases/{id}/reverse is the realistic blockchain remediation route once a public-chain transfer is final.

implemented
camt.056

Pre-broadcast cancellation only

Treated strictly as a pre-broadcast cancellation analogue. After broadcast the cancellation window is essentially zero on a public chain.

partial

Future investigation / exceptions & investigations analogues. Kept as reference-server draft machinery; the spec shape stays open until the design is clearer.
